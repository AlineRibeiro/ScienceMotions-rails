# README

To run the project:

1. git clone git@github.com:JessicaML/ScienceMotions-rails.git
2. cd ScienceMotions-rails
3. bundle install
4. rake db:create
5. rake db:migrate
6. rails server

if you are having problems creating the db, try 
- rake db:reset
- rake db:create
- rake db:migrate

* Ruby version

ruby 2.2.6
